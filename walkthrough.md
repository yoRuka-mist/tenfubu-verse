# 進化・超進化システム調整とバグ修正

## 実装内容

### 1. 進化回数制限 (1ターンに1回)
- 1ターンにつき、進化または超進化は1回のみに制限しました。

### 2. 進化解禁ターンの変更
- **通常進化**: 先攻5ターン / 後攻4ターン
- **超進化**: 先攻7ターン / 後攻6ターン

### 3. 進化可能通知機能
- 進化・超進化が可能なターンの開始時に、画面中央に通知アニメーションを表示するようにしました。
- **変更点**: 当初は「解禁された最初のターンのみ」表示していましたが、見逃し防止や分かりやすさのため、**「進化可能な条件を満たしているターンであれば毎回」** 表示するように仕様変更しました（ただし進化権がない場合は表示されません）。

### 4. その他バグ修正
- **エフェクト座標ズレ**: 解像度変更時やウィンドウサイズ変更時に、ダメージやスペルのエフェクト座標がずれる問題を修正しました。
- **カード操作系**: 素早い操作での情報消失や、操作不能になるバグを修正しました。

## 確認事項
- 指定ターン（先攻5T/後攻4T）になるまで進化できないこと。
- 1回進化すると、そのターンの間は進化ボタン（ドラッグ）が機能しなくなること。
- 進化可能なターンになったら、画面中央に通知が表示されること（毎ターン）。
