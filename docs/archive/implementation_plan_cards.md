3. 新カード「クレイジー・ナックラーズ」の実装

    - コスト: 5, スペル

4. 「侍茶」の効果変更
    - `id: s_samurai_tea`
    - 効果：「1枚ドローする。自分のフォロワーすべてを+1/+1する」に変更。
5. 「ウララ」の効果変更
    - `id: c_urara`
    - ステータス: 2/3 (体力+1)
